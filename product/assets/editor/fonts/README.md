This directory contains editor UI font assets.

Default editor font stack:
- Primary Latin UI font: `IBMPlexSans-Regular.ttf`
- Strong Latin UI font: `IBMPlexSans-SemiBold.ttf`
- Primary CJK merge font: `NotoSansSC-Regular.otf`
- Strong CJK merge font: `NotoSansSC-Medium.otf`
- `IBMPlexSans-Regular.ttf` license: see `IBMPlexSans-LICENSE.txt`
- `IBMPlexSans-SemiBold.ttf` license: see `IBMPlexSans-LICENSE.txt`
- `NotoSansSC-Regular.otf` license: see `LICENSE.txt`
- `NotoSansSC-Medium.otf` license: see `LICENSE.txt`

The runtime editor is configured from `EditorSettings.json`:
- `uiFontPath`
- `uiFontMergePath`
- `uiStrongFontPath`
- `uiStrongFontMergePath`
- `uiFontSizePixels`
- `uiUseFullChineseGlyphRange`
